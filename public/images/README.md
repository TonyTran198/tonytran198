# Profile Image

Đặt ảnh đại diện của bạn vào đây với tên file: `profile.jpg`

## Yêu cầu:
- Tên file: `profile.jpg` (hoặc `profile.png`)
- Kích thước khuyến nghị: 512x512px hoặc lớn hơn (tỷ lệ 1:1)
- Format: JPG hoặc PNG
- File size: Tối ưu dưới 500KB để tải nhanh

## Cách thêm ảnh:
1. Đặt ảnh của bạn vào thư mục này với tên `profile.jpg`
2. Hoặc sửa đường dẫn trong `components/About.tsx` nếu bạn dùng tên file khác

## Lưu ý:
- Ảnh sẽ được hiển thị dạng tròn (circular)
- Next.js sẽ tự động optimize ảnh
- Đảm bảo ảnh có chất lượng tốt và rõ nét
